**Team:** Turing Titans
**Course:** CS-AI-2025 — Building AI-Powered Applications
**Semester:** Spring 2026

---

## 🚀 Overview

**Film Adviser** is an AI-powered web application designed to help couples quickly find movies they will both enjoy.

Instead of endlessly scrolling through streaming platforms or arguing over choices, users input their preferences and receive **fair, balanced movie recommendations** generated by AI.

---

## ❗ Problem

Couples often struggle to choose a movie together because:

- Their tastes conflict
- Streaming platforms do not balance preferences fairly
- Decision-making takes too long (20–30 minutes on average)

This leads to frustration, indecision, and poor viewing choices.

---

## 💡 Solution

Film Adviser solves this by:

- Collecting preferences from both users
- Fetching candidate movies from TMDb
- Using AI to:
  - Score each movie for both users
  - Compute a **fairness-balanced score**
  - Generate short explanations

The result:
👉 A curated list of movies both users are likely to enjoy.

---

## ✨ Key Features

- 🤖 **AI-Powered Recommendation Engine**
  Balances two users’ preferences using intelligent scoring

- ⚖️ **Fairness Scoring System**
  Ensures one user’s taste does not dominate

- 🧠 **Explanation Generation**
  Provides 1–2 sentence reasoning for each recommendation

- 🔄 **Fallback Recommendation System**
  Works even if AI fails or returns invalid output

- 🎴 **Swipe-Based Interface**
  Tinder-style interaction for intuitive selection

---

## 🏗️ Architecture

### Frontend

- React
- Tailwind CSS
- Hosted on Vercel

### Backend

- FastAPI (Python)
- Hosted on Railway (EU region)

### AI Integration

- Primary Model: `openai/gpt-4.1-mini` (via OpenRouter)
- Fallback Model: `gemini 3 flash`

### Database

- Supabase PostgreSQL (EU region)
- Stores:
  - User preferences
  - Swipe history
  - Fairness logs

---

## 🔄 How It Works

1. **Users input preferences**
2. **System fetches ~50 candidate movies** from TMDb
3. **AI processes data and returns:**
   - Scores for each user
   - Fairness score
   - Explanation

4. **Top 5 movies are displayed** as swipeable cards

---

## ⚠️ Fallback Logic

If AI:

- ❌ Fails
- ⚠️ Returns invalid JSON

The system:

- Switches to deterministic scoring (average-based)
- Displays recommendations without explanations
- Shows a user-friendly message (no technical errors)

## 🛠️ Setup Instructions

### 1. Clone the repository
